# Agent Handoff Playbook

Use this when one human or agent needs to transfer work to another.

## Handoff Format

```text
Goal:
Current state:
Files changed:
Decisions made:
Checks run:
Known risks:
Next recommended step:
```

## Rules

- Be specific about file paths.
- Separate facts from assumptions.
- Include what was not done.
- Mention blocked work directly.
- Keep the handoff short enough to read before acting.

## Quality Bar

A receiving agent should be able to continue without rereading the entire conversation.
