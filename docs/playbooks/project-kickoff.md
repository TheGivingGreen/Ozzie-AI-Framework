# Project Kickoff Playbook

Use this when starting a new project that adopts the Ozzie AI Framework.

## Steps

1. Copy the framework files into the new project.
2. Fill out `templates/project-brief.md` in the target project.
3. Identify the project type: software, marketing, creative, operations, or hybrid.
4. Record constraints, success criteria, and non-goals.
5. Decide which agents or tools will participate.
6. Create the first scoped task using `templates/agent-task.md`.
7. Begin implementation only after the project-specific repository is clearly separated from this framework repository.

## Output

The kickoff should produce:

- A completed project brief
- A first task brief
- Known constraints
- Initial risks
- A clear owner for next action
