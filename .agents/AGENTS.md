# Agent Adapter

This is a compatibility entry point for agent tools that automatically read `.agents/AGENTS.md`.

Canonical instructions live at the repository root. Read these files before acting:

1. `README.md`
2. `OPERATING_SYSTEM.md`
3. `AGENT_PROTOCOL.md`
4. `GOVERNANCE.md`
5. `REPOSITORY_STANDARD.md`

Do not treat this adapter as a separate source of truth.
